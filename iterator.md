### 容器与迭代器  


#### 容器  

|  容器   | 描述  |
|  :----: | :---- |
|  Vec        |  可变长数组 连续存储  |
|  VecDeque   |  双向队列 适用于从头部或尾部删除插入数据  |
| LinkedList  |  双向链表 非连续存储  |
| HashMap     |  基于 Hash 算法存储一系列键值对 |
| BTreeMap    |  基于 B 树存储一系列键值对 |
| HashSet     |  基于 Hash 算法的集合 相当于没有值的 HashMap |
| BTreeSet    |  基于 B 树的集合 相当于没有值的 BTreeMap |
| BinaryHeap  |  基于二叉堆实现的优先级队列 |
